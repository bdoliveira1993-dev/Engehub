# 🏗️ Engehub

Plataforma web para centralização de oportunidades de prestação de serviços na engenharia.

🔗 Demo: https://engehub.vercel.app/  
📦 Repositório: https://github.com/bdoliveira1993-dev/Engehub

---

## 💡 Problema

Grande parte das oportunidades na engenharia (laudos, projetos, consultorias) não está em plataformas estruturadas.

Elas acontecem via:
- indicações
- grupos informais
- redes pessoais

Isso gera um mercado:
- fragmentado
- pouco acessível
- dependente de networking fechado

---

## ✅ Solução

O **Engehub** propõe uma centralização dessas oportunidades em uma única plataforma, permitindo que engenheiros encontrem e acompanhem demandas de forma mais eficiente e transparente.

---

## 🎯 Proposta de valor

- 📌 Centralizar oportunidades de engenharia
- 🔎 Facilitar descoberta de demandas
- 🤝 Reduzir dependência exclusiva de indicação
- ⚡ Tornar o acesso a trabalhos mais democrático

---

## 🧠 O que este projeto demonstra

Este projeto foi desenvolvido com foco em problemas reais de produto e boas práticas de engenharia:

- **Pensamento de produto orientado a mercado**
- **Modelagem de dashboard com dados dinâmicos**
- **Arquitetura preparada para evolução (ex: marketplace)**
- **Integração com backend (ex: Supabase / dados em tempo real)**
- **Construção de interfaces modernas e responsivas**

---

## 🖥️ Interface

A aplicação simula um ambiente real de uso, incluindo:

- Dashboard com métricas (laudos, projetos, networking)
- Listagem de oportunidades recentes
- Navegação por categorias:
  - Laudos e ARTs
  - Projetos
  - Consultorias
  - Empregos

---

## 🛠️ Stack técnica

- **Frontend:**  React + Vite (TypeScript)
- **Backend / DB:** Supabase
- **Estilização:** CSS customizado
- **Deploy:** Vercel

---

## ⚙️ Decisões técnicas

- Estrutura pensada para evolução para **marketplace**
- Organização modular para facilitar manutenção
- Uso de dados simulados + estrutura pronta para dados reais
- Interface otimizada para **rápida leitura e ação**
